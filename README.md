# EU GDPR Question Answering System using Retrieval-Augmented Generation (RAG)

This project implements a Retrieval-Augmented Generation (RAG) pipeline
to answer user queries from the official EU GDPR document.

Dataset source: EUR-Lex (Regulation EU 2016/679)
